Encabezado

Muchos desarrolladores de páginas web emplean las etiquetas "div" y "spam" para definir encabezados (títulos) de sección y estructurar el contenido. Sin embargo, estas etiquetas no proporcionan contenido semántico, por lo que se debe evitar su uso. Lo correcto es utilizar las etiquetas de encabezado "h1" ... "h6" y definir su presentación visual con CSS.

Elementos clave de la solución

1. Correcto uso de las etiquetas de XHTML para definir encabezados: "h1" ... "h6".

2. No utilización de las etiquetas sin contenido semántico de XHTML 2div" y "span".

3. Correcto empleo de las propiedades CSS para definir la presentación de los encabezados: font-size y font-weight.
